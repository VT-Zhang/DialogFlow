## Strayer University Coding Challenge project.
### 1. Please run your MongoDB server first before you start the node server.<br>
####    ``` $ mongod```<br>  
### 2. Install the dependencies<br>
####    ``` $ npm install```<br>
### 3. Run the server<br>
####    ``` $ npm start```<br>
### 4. Open the browser and go to<br>
####    ``` http://localhost:5000```<br>
