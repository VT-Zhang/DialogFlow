## This is for Strayer University Chatbot Coding Challenge project.
1. Please run your MongoDB server first before you start the node serve.<br>
2. Install<br>
    ``` $ npm install          ```<br>
3. Run the server<br>
    ``` $ npm start            ```<br>
4. Open the browser and go to<br>
    ``` http://localhost:5000  ```<br>
